| **参数** | **参数意义**                                     | **用法示例**                               |
| :------- | :----------------------------------------------- | :----------------------------------------- |
| -i       | 带一个参数，表示输入文件的路径（绝对或相对路径） | `intersect.exe -i input.txt -o output.txt` |
| -o       | 带一个参数，表示输出文件的路径（绝对或相对路径） | `intersect.exe -i input.txt -o output.txt` |
